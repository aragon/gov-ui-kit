<Title>OdsCoreProvider</Title>

The `<OdsCoreProvider />` is a React context that defines the image and link elements to be rendered across all the ODS
components, defaulting such values to `<img />` and `<a />` tags. The provider is used by the `<AvatarBase />` and
`<LinkBase />` components, which are in turn used by all the other core and module components.

The provider becomes particularly useful when an application needs to employ framework-specific images or links, such as
the `<Image />` and `<Link />` components from Next.js.

**Usage example**

```typescript
import { OdsCoreProvider } from '@aragon-ods';
import NextImage from 'next/image';
import NextLink from 'next/link';

const CustomImage: React.FC<ComponentProps<'img'>> = ({ src = '', alt = 'image', width, height, ...otherProps }) => (
    <NextImage src={src} fill={true} alt={alt} {...otherProps} />
);

const CustomLink: React.FC<ComponentProps<'a'>> = ({ href = {}, ...otherProps }) => (
    <NextLink href={href} {...otherProps} />
);

const coreProviderValues = { Img: CustomImage, Link: CustomLink };

export const Application = () => {
    return (
        <OdsCoreProvider values={coreProviderValues}>
            /* Your application */
        </OdsCoreProvider>
    )
}
```

## Localization

The `copy` property in the `OdsCoreProvider` provides localized text copy for the core components. It is an object that
adheres to the `IOdsCoreCopy` interface, which includes all the text strings used within the ODS core components. This
allows for easy localization of the application by switching the `copy` object based on the user's language preference.

The ODS library provides copies for a few popular languages. You can implement the `IOdsCoreCopy` interface for any
language and provide it as a prop to the provider.

**Usage example**

```tsx
import { OdsCoreProvider, esCopy } from '@aragon-ods';

const coreProviderValues = { copy: esCopy };

export const Application = () => {
    return <OdsCoreProvider values={coreProviderValues}>/* Your application */</OdsCoreProvider>;
};
```
