# Installation

Install the `@aragon/gov-ui-kit` library and using Yarn or npm:

yarn:

```shell
yarn add @aragon/gov-ui-kit
```

npm:

```shell
npm install @aragon/gov-ui-kit
```

# Setup

You can setup the Aragon Governance UI Kit library in your project with [Taildwind CSS](https://tailwindcss.com) or as a
standalone library:

- Follow the [Setup with TailwindCSS](#setup-with-tailwindcss) guide if you are using or planning to use `Tailwind CSS`
  for your project.
- Follow the [Standalone Setup](#standalone-setup) guide if you are using your own or another UI library.

## Setup with Tailwind CSS

- Setup `Tailwind CSS` for your project by following their
  [installation guide](https://tailwindcss.com/docs/installation)

- Install the required `TailwindCSS` plugins:

    ```shell
    yarn add @tailwindcss/typography
    ```

    or

    ```shell
    npm install @tailwindcss/typography
    ```

- In your main CSS file, import the library theme and use the `@source` directive to ensure TailwindCSS detects all
  classes used by the library:

    ```css
    @import 'tailwindcss';
    @import '../node_modules/@aragon/gov-ui-kit/index.css';
    @source '../node_modules/@aragon/gov-ui-kit';
    ```

    **NOTE**: Adjust the import paths according to the actual location of the `node_modules` directory in your project
    structure.

## Standalone Setup

- Import the Aragon Governance UI Kit stylesheet bundle in your project entry file (e.g. `src/index.js` or `App.js`):

    ```typescript
    import '@aragon/gov-ui-kit/build.css';
    ```

# Usage

Import the `@aragon/gov-ui-kit` library on your application and start using the components:

```typescript
import React from 'react';
import { Button } from '@aragon/gov-ui-kit';

export const App: React.FC = () => (
    <div>
        <Button variant="primary" size="lg">My label</Tag>
    </div>
}
```
